# Todo List App

A modern, responsive todo list application built with vanilla JavaScript, HTML5, and CSS3. Features a clean, intuitive interface with local storage persistence, filtering capabilities, and a beautiful gradient design.

## 🚀 Features

- ✅ **Add Tasks**: Quickly add new todo items with a clean input interface
- ✅ **Mark Complete/Incomplete**: Toggle task completion status with smooth animations
- ✅ **Edit Tasks**: Inline editing with Enter/Escape key support
- ✅ **Delete Tasks**: Remove tasks with confirmation dialog
- ✅ **Filter Tasks**: View All, Active, or Completed tasks
- ✅ **Clear Completed**: Bulk remove completed tasks
- ✅ **Local Storage**: Tasks persist between browser sessions
- ✅ **Statistics**: Real-time task count statistics
- ✅ **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- ✅ **Keyboard Shortcuts**: Ctrl+Enter to add tasks, Enter/Escape for editing
- ✅ **Notifications**: User-friendly toast notifications instead of alerts
- ✅ **Export/Import**: Export todos to JSON file (available in console)
- ✅ **Search Functionality**: Search through tasks (available in console)
- ✅ **Accessibility**: ARIA labels and keyboard navigation support

## 🎨 Design Features

The app is fully responsive and works seamlessly across all devices:

## 🎯 Usage

1. **Adding Tasks**: Type in the input field and click "Add Task" or press Ctrl+Enter
2. **Completing Tasks**: Click the checkbox next to any task
3. **Editing Tasks**: Click the edit icon (pencil) and modify the text
4. **Deleting Tasks**: Click the trash icon to remove a task
5. **Filtering**: Use the filter buttons to show different task categories
6. **Clearing Completed**: Click "Clear Completed" to remove all finished tasks

## 🔧 Advanced Features (Console)

Access these features through the browser console:

```javascript
// Export todos to JSON file
todoApp.exportTodos()

// Search todos
todoApp.searchTodos("search term")

// Access all todos
todoApp.todos
```
